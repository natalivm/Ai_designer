# AI for Product Designers

Онлайн-курс **AI for Product Designers** — програма для продуктових дизайнерів, UX-дослідників та аналітиків. Практичні розділи та інтерактивні квізи проведуть вас через повний продуктовий флоу: від AI-ресерчу і побудови персон до прототипів, UX-текстів і vibe-кодингу. Наприкінці ви зберете повноцінний кейс для портфоліо — AI-асистент для дизайну.

## Stack

- React 19 + Vite
- Tailwind CSS v4
- GitHub Pages deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

# AI for Product Designers

Лендінг онлайн-курсу **AI for Product Designers** — програми для продуктових дизайнерів, UX-дослідників та аналітиків. Живі вебінари та куратор проведуть вас через повний продуктовий флоу: від AI-ресерчу і побудови персон до прототипів, UX-текстів і vibe-кодингу. Наприкінці ви зберете повноцінний кейс для портфоліо — AI-асистент для дизайну.

## Stack

- React 19 + Vite
- Tailwind CSS v4
- GitHub Pages deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
